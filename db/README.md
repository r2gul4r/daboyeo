# Database

TiDB/MySQL 관련 파일을 모으는 영역이다.

현재 실제 SQL 이력은 기존 `scripts/db/`에 있다. 이 폴더는 협업용 DB 책임 경계를 명확하게 두기 위해 만든다.

## 디렉터리

- `sql/`: TiDB/MySQL에 직접 적용할 수 있는 수동 SQL과 관련 문서
- `seeds/`: 로컬 개발용 기준 데이터
- `samples/`: 공유 가능한 샘플 데이터 자리

## 원칙

- 운영 계정, 비밀번호, 접속 문자열은 커밋하지 않는다.
- TiDB가 MySQL 호환이라도 DDL은 단순하고 명시적으로 작성한다.
- 수집기 원본 데이터는 가능한 한 JSON으로 보존하고, 비교에 필요한 공통 필드만 정규화한다.
